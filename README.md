## Usage

```bash
git submodule add https://github.com/ahampriyanshu/hugle.git ./themes/hugle
```

```bash
git submodule update --remote --merge
```

## Development

```bash
brew install git node go hugo
git clone https://github.com/ahampriyanshu/hugle.git
cd hugle
yarn
yarn dev
```
